e para usar o itálico se usa 'em' ou 'i'

e para marcar se uso 'MARK'

e para que seja dada mais ênfase se usa 'INS' ou 'U de inserido'

para um uso mais matemático ou fórmulas químicas se usa 'SUP' para elevar algo e 'SUP' para abaixar: X2+3 (um sobe)
